# SaaS Localize

SaaS Localize is a Python project that integrates with popular CI/CD tools to automate the translation process for SaaS applications.

## Features

* Integrates with Jenkins and GitLab CI/CD tools
* Supports at least 5 language pairs
* Automates the translation workflow

## Usage

1. Install the project using `poetry install`
2. Run the project using `python -m saas_localize`
3. Pass the language pairs and CI/CD tool as arguments using `--language-pairs` and `--ci-cd-tool` flags

## Testing

1. Install the project using `poetry install`
2. Run the tests using `pytest`
